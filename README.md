# streamlit_data_analyze
Streamlitでデータ分析するためのWEBアプリ

変更前後で画像データの比較ができます。

実験結果の記録ツールとしてもご活用できます。

業務に合わせてご自由にアレンジしてみてください。

### Streamlitのインストール

pip install streamlit

### 実行方法

### 比較ツール

streamlit run compare_img.py

### pygwalker

streamlit run pygwalker_demo.py
